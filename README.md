# RO^JO

Fast, minimal BitTorrent client built with Electron and WebTorrent.

## Features

- **Drag & drop** `.torrent` files or magnet links
- **Per-torrent download path** — choose folder each time you add a torrent
- **Pause / Resume** individual torrents
- **Remove** from list (keep files) or **Delete** with files
- **Detail panel** with battery-style progress bar, peers, speed, ETA, ratio
- **Dock badge** shows live download speed when minimized (macOS)
- **Taskbar progress** shows completion percentage when minimized
- **Dark theme** UI with custom title bar
- Pre-configured optimal settings for best download experience
- Default `.torrent` file handler and `magnet:` protocol support
